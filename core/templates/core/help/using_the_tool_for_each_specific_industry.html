How to Use the Tool for Each Specific Industry?

Using the forecasting tool

  1. Begin by selecting a specific industry from the following options: Retail, Technology, Healthcare, Automotive, or Hospitality.
  2. Once an industry is selected, you'll be presented with a description and the necessary fields for that industry. Note that while each industry has its unique fields, all include Revenue and Date fields.
  3. Scroll further to find the Sample JSON Format and Sample CSV Format. You can utilize either format.
  4. Click the "Go to Forecasting Tool" button to proceed to the tool page for the selected industry.
  5. On this page, you will find:
    • Header - Your chosen industry (e.g., Retail).
    • Sales Forecast - This section displays forecasted monthly sales values.
    • Insights - Here, you'll see the total, highest, lowest, and average forecasted monthly sales.
    • Graph - A visualization of the forecasted monthly sales.
  6. Below the graph, there are three buttons: Upload, Process, and Reset, each with its own functionality.
  7. Click the "Upload" button and select a valid historical sales data file (either JSON or CSV). Here's what a valid data file should have:
    • At least 3 years of data.
    • Complete data for all months within those years. For instance, if your data spans from 2020-2022, it should contain records for all months of each year.
    • All the necessary fields filled in. Consult the industry's Sample Format for specifics.
  8. Click the Process button to run the Forecasting System. Once processed, the Sales Forecast, Insights, and Graph sections will update.
  9. Click the Reset button to clear all data and start over.
